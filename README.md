# Amazon Navbar Clone

A responsive Amazon-style navigation bar built using HTML and CSS.  
This project replicates the design and layout of Amazon’s navigation bar for learning front-end development.

## 🚀 Features
- Amazon-style navigation bar
- Responsive layout
- Clean UI design
- Organized HTML & CSS structure

## 🛠️ Technologies Used
- HTML5
- CSS3
- Flexbox
